Windows Setup

  1. Open Settings → Network & Internet
  2. Click your connection (Wi-Fi or Ethernet) → Properties
  3. Under IP settings, click Edit
  4. Switch to Manual, enable IPv4
  5. In Preferred DNS, enter: loading...
  6. In Alternate DNS: 1.1.1.1 (fallback)
  7. Save — you're protected!
You can also use 127.0.0.1 if S.O.W.A is running on this same computer.

Router Setup (Protect Entire Network)

  1. Open your router's admin panel (usually 192.168.0.1 or 192.168.1.1)
  2. Find DNS settings (usually under DHCP, WAN, or Internet)
  3. Set Primary DNS to: loading...
  4. Set Secondary DNS to: 1.1.1.1
  5. Save and restart the router if needed
This will protect all devices on your network — phones, tablets, smart TVs, etc.
